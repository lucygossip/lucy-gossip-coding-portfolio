This project was part of a frontent training course with Kreativstorm.

My group was tasked with creating a functionable calculator, using HTML, CSS and JavaScript. The calculator had to be able to perform basic operations, as well as have keyboard support.

One of my jobs was to create CSS to make the page responsive. I added design options based on user preferences, including light and dark mode, contrast and reduced motion.

I also tested the calculator and used my findings to provide feedback to the rest of the group. As a result, we were able to identify and fix bugs in the code.
